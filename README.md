## 1. Модификации программы

В процессе выполнения лабораторной работы базовый код был доработан и расширен.

### 1.1. Автоматический запуск серии тестов  

Программа выполняет вычисления для набора размеров матриц:

200, 400, 800, 1200, 1600, 2000, 2400, 2800, 3200, 3600.

Замеры времени для каждого размера автоматически сохраняются в файл `timings.txt`.

### 1.2. Раздельное сохранение данных  

Для каждого значения N формируются отдельные файлы:

- `A_N.txt` — первая матрица  
- `B_N.txt` — вторая матрица  
- `C_N.txt` — результат умножения  

Это позволяет избежать перезаписи данных и упрощает анализ результатов.

### 1.3. Повышенная точность вывода  

При записи матриц используется фиксированный формат с точностью 10 знаков после запятой, что позволяет сохранить все значащие цифры.

### 1.4. Проверка корректности результатов  

Дополнительно был использован Python-скрипт с библиотекой NumPy для проверки корректности вычислений.  

Скрипт вычисляет:
- максимальную абсолютную ошибку  
- максимальную относительную ошибку

## 2. Выполнение работы

### 2.1. Реализация алгоритма

Программа написана на языке C++.

Реализован классический алгоритм последовательного перемножения матриц со сложностью:

O(N³)

Для измерения времени выполнения использовалась библиотека `std::chrono`.

---

### 2.2. Проведение экспериментов

Для каждого значения N:

1. Генерируются матрицы A и B  
2. Выполняется их умножение  
3. Результат сохраняется в файл  
4. Фиксируется время выполнения  

Все результаты записываются в `timings.txt`.

---

### 2.3. Верификация

Проверка выполнялась в среде Google Colab с использованием Python и NumPy.

Вычислялись:
- максимальная абсолютная ошибка  
- максимальная относительная ошибка  

Полученные значения подтвердили корректность работы программы.
